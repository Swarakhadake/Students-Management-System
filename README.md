# 🎓 Student Management System

This is a Python-based GUI application that helps manage student records efficiently using a simple interface.

## 🚀 Features

* Add new student details
* Update existing student records
* Delete student data
* Search students by name
* Display all student records
* Easy-to-use graphical interface

## 🛠️ Technologies Used

* Python
* Tkinter
* SQLite3

## Output Photo
* Output.png file

## ▶️ How to Run

1. Install Python
2. Run the file:
   python app.py
